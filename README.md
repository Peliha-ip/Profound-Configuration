# Profound-Configuration
This is a Minecraft mod that adds the long-awaited Deep Dark dimension, alongside the Cyber Dimension.

Mobs:
- Shade

Blocks:
- Sculk jaw

Items:
-Shade spawn egg

Features: 
- Sculk jaws trap anything that walks on it and damage it over time. Repeatedly jumping will cause the jaw to relax and let whatever was trapped in it escape.
